= App Feelpp Distance

In this mini-app, we take a cubic geometry, and compute the distance of each node to the border of the geometry.

Run the application by hand:

[source, bash]
----
feelpp_app_distance [--h <h> --M <M> --gmsh.filename <path>]
----

- `h` is the size of the mesh
- `M` is the number of ray casted for the BVH method
- `gmsh.filename` is the path to the geometry (or the mesh). This geometry should contain a physical marker named `boundary`.
